Battleship Game


Overview
A text-based Battleship game in C++ where you play against an AI using a hunt-and-target strategy. The game includes board setup, ship placement, hit/miss tracking, and round-based turns.


Features
        •        Ship placement and board setup.
        •        AI with intelligent targeting.
        •        Turn-based gameplay with hit/miss feedback.
        •        Win when all opponent ships are sunk.


How to Play


1. Run the code
2. Place your ships by entering coordinates and orientation (H for horizontal, V for vertical).
3. Take turns firing at the opponent’s board.
4. The game ends when all ships of one player are sunk.
